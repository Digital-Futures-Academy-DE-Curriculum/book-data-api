Books to Scrape API

This is a RESTful API using information previously scraped book information from the Books to Scrape website.

The routes available in this API are:

  • /categories
    • Returns the whole collection of categories with all of the stored data
  • /categories/books
    • Returns an object where the keys are the category names and the values are an object that contains the number of books in the category and an array of all of the books in the category
  • /categories/books/count
    • Returns an object with keys of the category names and the values are the number of books in the category
  • /categories/books/prices
    • Returns an object with keys of the category names and values of the number of books and an array that contains the price of each book
  • /category/<category_id>
    • Returns an object with a key of the category name and the value of the number of books - this will match the category that has the id supplied
    • In this case 672e3538a2a5c1a72454d835 which is the ID for Philosophy
  • /categories/<category_id>/books
    • Returns an object with a key that represents the category name and an object that contains the number of books in the category and an array of the books in the category
    • In this case 672e3538a2a5c1a72454d835 which is the ID for Philosophy
  • /categories/<category_id>/books/prices
    • Returns an object with a key that represents the category name and an object that contains the number of books in the category and an array of the book prices in the category
    • In this case 672e3538a2a5c1a72454d835 which is the ID for Philosophy
